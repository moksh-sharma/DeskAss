# Microsoft Office Performance Problems

## Symptoms
- Word/Excel/PowerPoint slow to open or save.
- Lag while typing or scrolling.
- Freezes on large documents/workbooks.

## Common Root Causes
- Problematic add-ins.
- Hardware acceleration / graphics driver issues.
- Very large or complex files.
- Slow network location for the file.
- Outdated Office build.

## Recommended Fixes (require user confirmation)
1. Start the app in Safe Mode (`winword /safe`) to test add-ins.
2. Disable unnecessary COM add-ins.
3. Turn off hardware graphics acceleration: File > Options > Advanced.
4. Copy network files locally before editing large documents.
5. Update Office and graphics drivers.
6. Run an Office repair.

## Prevention
- Keep Office and drivers updated.
- Avoid editing very large files directly over the network.
- Limit add-ins.

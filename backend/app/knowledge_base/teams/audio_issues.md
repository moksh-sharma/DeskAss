# Microsoft Teams Audio / Microphone Issues

## Symptoms
- No sound, or others can't hear you.
- Microphone not detected.
- Echo or robotic audio.

## Common Root Causes
- Wrong default audio device selected.
- Windows privacy settings blocking microphone access.
- Outdated audio drivers.
- Exclusive-mode apps holding the device.

## Recommended Fixes (require user confirmation)
1. Teams > Settings > Devices: select the correct speaker/microphone and make a test call.
2. Windows Settings > Privacy > Microphone: ensure access is enabled for apps.
3. Set the correct default device in Windows Sound settings.
4. Update or reinstall audio drivers.
5. Restart Teams and the Windows Audio service.

## Prevention
- Standardise on a known-good headset.
- Keep audio drivers updated.
